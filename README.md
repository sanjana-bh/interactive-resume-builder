# Interactive Resume Builder

Interactive Resume Builder is a web application that allows users to create professional resumes easily using a simple and interactive interface.

## Features
- Dynamic resume form
- Real-time preview
- Clean and responsive design
- Easy resume customization

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Use
1. Enter your personal and professional details.
2. Preview your resume instantly.
3. Customize the layout and information.

This project demonstrates front-end development skills and dynamic user interface design.
